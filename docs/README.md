# Dragon Ecosystem Documentation

These documents describe the Slice 0.1 foundation state only. They record locked direction and current project boundaries without adding product feature specifications or future implementation details.

## Sections

- `adr/` — architecture decision records and HQ-locked direction placeholders.
- `architecture/` — current Slice 0.1 architecture overview.
- `development/` — local development setup and environment conventions.
- `operations/` — operations placeholder documentation for later slices.
- `security/` — security placeholder documentation for later slices.

Slice 0.1 is limited to monorepo foundation, minimal app/package shells, local-only infrastructure, environment examples, CI, and base documentation. Product features are not implemented yet.
