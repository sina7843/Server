# ADR 0005 — Single VPS Docker Compose

## Status

Accepted / Locked by HQ

## Context

Dragon Ecosystem has a locked future deployment direction based on a single VPS and Docker Compose.

## Decision

Single VPS with Docker Compose is a locked future deployment direction.

## Scope in Slice 0.1

Slice 0.1 includes local Docker Compose infrastructure only. It does not represent production deployment.

## Not implemented yet

No production deployment, Arvan VPS setup, production compose, SSL, backup, monitoring, or incident runbook is implemented in Slice 0.1. Implementation comes in later slices.
