# Dragon Ecosystem

Phase 0 / Slice 0.1 initializes the empty monorepo foundation for Dragon Ecosystem.

This repository uses pnpm workspaces with apps under `apps/*` and shared packages under `packages/*`.

Monorepo task orchestration is handled by Turborepo through root pnpm scripts.

Root tooling commands include linting, type checking, testing, building, and Prettier format checks.

No product features are implemented yet.
