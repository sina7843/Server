# ADR 0003 — RBAC/ABAC Ready Authorization

## Status

Accepted / Locked by HQ

## Context

Dragon Ecosystem must be ready for authorization requirements that may include role-based and attribute-based rules in later slices.

## Decision

Authorization must be RBAC-ready and ABAC-ready later.

## Scope in Slice 0.1

No roles, permissions, policies, guards, decorators, or admin permission UI exist in Slice 0.1.

## Not implemented yet

Authorization implementation comes in later slices. This ADR does not define a permission model, role hierarchy, policy structure, guard system, or admin permission workflow.
