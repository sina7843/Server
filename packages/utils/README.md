# @dragon/utils

This package contains shared generic Dragon utilities only.

It must not contain product feature logic. It must not contain database, queue, API client, or UI logic.

Slice 0.1 only includes neutral foundation-level utilities.

## Phone utilities

Slice 0.2 adds generic, framework-agnostic phone normalization helpers. These helpers are not tied to Auth flows, SMS providers, country-specific UI behavior, or product features.
