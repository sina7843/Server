# ADR 0002 — Phone-First Auth

## Status

Accepted / Locked by HQ

## Context

Phone-first authentication is a locked future product direction for Dragon Ecosystem.

## Decision

Dragon Ecosystem will support phone-first Auth in a later slice.

## Scope in Slice 0.1

Slice 0.1 contains no Auth implementation. No OTP, SMS, session, token, or user implementation exists in Slice 0.1.

## Not implemented yet

Auth implementation comes in later slices. This ADR does not define OTP flow, user schema, token strategy, session strategy, or provider integration.
