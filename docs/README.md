# Dragon Ecosystem Documentation

These documents describe the current approved foundation state of Dragon Ecosystem. Slice 0.1 covers monorepo and infrastructure foundation. Slice 0.2 covers the backend Auth / OTP / Session foundation.

## Sections

- `adr/` — architecture decision records and HQ-locked direction.
- `architecture/` — current architecture overviews, including Auth architecture.
- `development/` — local setup, environment conventions, and slice verification checklists.
- `operations/` — operations placeholder documentation for later slices.
- `security/` — security documentation and checklists.

## Auth documentation

- `architecture/auth.md` — Slice 0.2 Auth architecture.
- `security/auth-security-checklist.md` — Slice 0.2 Auth security checklist.
- `development/slice-0.2-verification.md` — final Slice 0.2 verification checklist.

Slice 0.2 does not implement RBAC, Profile, Admin features, real SMS provider integration, Nuxt Auth UI, production deployment, monitoring, or backup.
