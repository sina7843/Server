# @dragon/config

Shared base tooling configuration for Dragon Ecosystem.

Included configuration foundations:

- TypeScript base, Node, and Vue tsconfig variants
- ESLint shared base config
- Prettier shared config
- Tailwind preset foundation

Root tooling can reuse these shared configuration files to keep the monorepo baseline consistent.

This package contains shared tooling config only. Product feature logic, app-specific configuration, UI components, and product design tokens do not belong here.
